# Medium Clone App

Here it is a full implementation of a Medium Clone App developed using NextJS, TailwindCSS and Firebase.

## How to try the app

First, install the dependencies:

```bash
npm install
# or
yarn install
```

Then, run the server:

```bash
npm run dev
# or
yarn dev
```

You will access the app in http://localhost:3000/

Enjoy coding!
